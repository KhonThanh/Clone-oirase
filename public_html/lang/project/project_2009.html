～美しいブナ林の紅葉と渓流のせせらぎを歩く～
奥入瀬渓流全区間１４ｋｍ（三里半）、約４～５時間のコースです。
  • 実施日／２００９年１０月３１日、１１月１日
  • 集合場所／奥入瀬渓流館／１０：００スタート
  • コース／奥入瀬渓流館（焼山） → 奥入瀬渓流１４ｋｍ → 子ノ口（遊覧船） → 遊覧船移動 → 休屋
  • 申し込み期限／１０月２６日（月）
